## B. Address translation
```
+--------------------+-------+
|     Parameter      | Value |
+--------------------+-------+
| VPN                | 9     |
| TLB index          | 1     |
| TLB tag            | 2     |
| TLB hit (Y/N)      | N     |
| Page fault (Y/N)   | N     |
| PPN                | 17    |
+--------------------+-------+
```

## C. Physical address format

PPO = VPO = 0b111100
